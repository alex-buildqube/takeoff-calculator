Implement **RFC-009: Centroid Reposition (Measurement Move by Centroid)** in strict accordance with the RFC document.

## Purpose

Add an API that, given a measurement and a new centroid, returns the measurement with its geometry translated so the centroid lies at the given point, enabling end users to drag a centroid and update the position of the measurement in consumer apps.
